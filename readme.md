# WebFetch Explorer

This tool is mainly aimed for developers during backend operations. This website allows API fetching with GET/POST/PUT/DELETE parameters.

## Switch behavior

### Read-only function

Displays the API operations (GET, POST, PUT DELETE). An user can copy-paste an endpoint and execute operations while watching the responses. A JSON file can be generated after a request if an json/text file is expected.

### Stress Mode

An Stress Test is executed under an endpoint in order to measure the origin latency. A pdf document can be generated with the different GET attempts over an endpoint.
